Class: Factory

core/factory~ Factory

new Factory(ctorOrOptions, options)

Factory class for creating arbitrary classes.
Parameters:
Name Type Argument Description
ctorOrOptions constructor | object Class constructor or object literal with 'ctor' attribute
options object <optional>
 Options to pass to ctor. Note that additional options can be added at creation time through module:core/factory~Factory#create This parameter is not required if ctorOrOptions is an object literal.

Examples
var factory = new Factory(MyConstructor, {
  option1: 'hello'  
});
var instance = factory.create({
  options2: 'hello2'
